# Dual-Mode Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Make Toggle and Push-to-Talk modes simultaneously active, removing the mode dropdown.

**Architecture:** Replace `_isPttMode` bool with a `_recordingSource` enum that tracks which shortcut started the current recording. Both event handlers are always active but ignore events when a recording from the other source is in progress.

**Tech Stack:** WPF, C# (.NET 8)

---

### Task 1: Add RecordingSource enum and replace _isPttMode

**Files:**
- Modify: `MainWindow.xaml.cs:35` (replace `_isPttMode` field)
- Modify: `MainWindow.xaml.cs:794-816` (update event handlers)

**Step 1: Add enum and replace field**

At the top of the `MainWindow` class (around line 35), replace:

```csharp

```

with:

```csharp
private enum RecordingSource { None, Toggle, Ptt }
private RecordingSource _recordingSource = RecordingSource.None;
```

**Step 2: Update OnToggleHotkey (line 794)**

Replace:

```csharp
private void OnToggleHotkey()
{
    if (_isPttMode) return;
    Dispatcher.Invoke(() =>
    {
        if (_recording) StopRecording();
        else            StartRecording();
    });
}
```

with:

```csharp
private void OnToggleHotkey()
{
    Dispatcher.Invoke(() =>
    {
        if (_recording)
        {
            if (_recordingSource != RecordingSource.Toggle) return;
            _recordingSource = RecordingSource.None;
            StopRecording();
        }
        else
        {
            _recordingSource = RecordingSource.Toggle;
            StartRecording();
        }
    });
}
```

**Step 3: Update OnPttKeyDown (line 806)**

Replace:

```csharp

{

    Dispatcher.Invoke(StartRecording);
}
```

with:

```csharp
private void OnPttKeyDown()
{
    if (!_connected || _recording) return;
    Dispatcher.Invoke(() =>
    {
        _recordingSource = RecordingSource.Ptt;
        StartRecording();
    });
}
```

**Step 4: Update OnPttKeyUp (line 812)**

Replace:

```csharp
private void OnPttKeyUp()
{
    if (!_isPttMode) return;
    Dispatcher.Invoke(StopRecording);
}
```

with:

```csharp
private void OnPttKeyUp()
{
    if (_recordingSource != RecordingSource.Ptt) return;
    Dispatcher.Invoke(() =>
    {
        _recordingSource = RecordingSource.None;
        StopRecording();
    });
}
```

**Step 5: Build and verify**
